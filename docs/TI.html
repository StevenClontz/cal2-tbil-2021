Chapter 1 Techniques of Integration (TI)

Learning Outcomes

How do we use various techniques to integrate less simple functions?

By the end of this chapter, you should be able to...

  1. Evaluate various integrals via the substitution method.

  2. Compute integrals using integration by parts.

  3. Compute integrals involving products of trigonometric functions.

  4. Use trigonometric substitution to compute indefinite integrals.

  5. I can integrate functions using a table of integrals.

  6. I can integrate functions using the method of partial fractions.

  7. I can select appropriate strategies for integration.

  8. I can compute improper integrals.
