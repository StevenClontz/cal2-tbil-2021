Activity 1.4.1

Consider the integral \(\displaystyle \int \frac{\sqrt{1-x^2}}{x} dx\text{.}\) If one were to use the method of substitution to evaluate this integral, what would be the best choice of \(u\text{?}\)

  1. \(\displaystyle u=1-x^2.\)
  2. \(\displaystyle u=\sqrt{1-x^2}.\)
  3. \(\displaystyle u=\frac{1}{x}.\)
  4. There is no u for which the method of substitution will work.

Activity 1.4.2

It would be good if \(\sqrt{1-x^2}\) were a term that can be written more simply. Which of the following mathematical facts could help us simplify this integrand?

  1. For any real number, \(x^2\) must be positive (note: this isn’t even true).
  2. \(\displaystyle u=\sqrt{1-x^2}.\)
  3. The Pythagorean theorem allows us to write \((\cos(\theta))^2\) as \(1-(\sin(\theta))^2\text{.}\)
  4. The ratio of an even function and an odd function is odd.

Activity 1.4.3

Suppose we let \(x=\sin(\theta)\text{,}\) then \(\frac{dx}{d\theta}=\cos(\theta)\) and \(dx=\cos(\theta) d\theta\text{.}\) Which of the following best represents the integral \(\displaystyle \int \sqrt{1-x^2}dx\text{?}\)

  1. \(\displaystyle \int \sqrt{1-\sin^2(\theta)}dx\text{.}\)
  2. \(\displaystyle \int \sqrt{ 1-\sin^2(\theta)\cos(\theta)}d\theta\text{.}\)
  3. \(\displaystyle \int \sqrt{ 1-\sin^2(\theta)}\cos(\theta)d\theta\text{.}\)

Activity 1.4.8

Consider the integral \(\displaystyle \int \sqrt{9-x^2} dx\text{;}\) how can we best modify our approach in Activity 1.4.3 and adapt it here?

  1. Let \(x=\sin(\theta), dx=\cos(\theta)d\theta, \displaystyle \int \sqrt{9-x^2} dx= \int \sqrt{9-\sin^2(\theta)} \cos(\theta) d\theta.\)
  2. Let \(x=3\sin(\theta), dx=3\cos(\theta)d\theta, \displaystyle \int \sqrt{9-x^2} dx= \int \sqrt{9-(3\sin(\theta))^2} \cos(\theta) d\theta.\)
  3. Let \(x=\sin(3\theta), dx=3\cos(3\theta)d\theta, \displaystyle \int \sqrt{9-x^2} dx= \int \sqrt{9-\sin^2(3\theta)} 3\cos(3\theta) d\theta.\)

Activity 1.4.11

Consider the integral \(\displaystyle \int \frac{1}{1+x^2} dx\text{.}\) Which of the following facts would best simplify this expression?

  1. \(\cos^2(\theta) =\frac{1+\cos(2\theta)}{2}\text{.}\)
  2. \(\displaystyle (\cos(\theta))^2=1-(\sin(\theta))^2\)
  3. Since \(\sin^2(\theta) + \cos^2(\theta)=1, \tan^2(\theta)+1=\sec^2(\theta)\text{.}\)

  2. \(\displaystyle \int \frac{1}{1+\tan^2(\theta)\sec^2(\theta)} d\theta\text{.}\)
  3. \(\displaystyle \int \frac{1}{1+\tan^2(\theta)} \sec^2(\theta)d\theta\text{.}\)

Activity 1.4.14

Consider the integral \(\displaystyle \int \frac{1}{\frac{1}{4}+x^2} dx\text{;}\) how can we best modify our approach in Activity 1.4.12 and adapt it here?

  1. Let \(x=\frac{1}{2}\tan(\theta), dx=\frac{1}{2}\sec^2(\theta)d\theta, \displaystyle \int \frac{1}{\frac{1}{4}+x^2} dx = \int \frac{1}{\frac{1}{4}+\left(\frac{1}{2}\tan(\theta) \right)^2} \frac{1}{2} \sec^2(\theta)d\theta.\)
  2. Let \(x=\frac{1}{4}\tan(\theta), dx=\frac{1}{4}\sec^2(\theta)d\theta, \displaystyle \int \frac{1}{\frac{1}{4}+x^2} dx = \int \frac{1}{\frac{1}{4}+\left(\frac{1}{4}\tan(\theta) \right)^2} \frac{1}{4}\sec^2(\theta) d\theta.\)
  3. Let \(x=\tan\left(\frac{1}{2}\theta\right), dx=\sec^2\left(\frac{1}{2}\theta\right)d\theta, \displaystyle \int \frac{1}{\frac{1}{4}+x^2} dx = \int \frac{1}{\frac{1}{4}+\left(\tan\left(\frac{1}{2}\theta\right) \right)^2} \frac{1}{2} \sec^2\left(\frac{1}{2}\theta\right)d\theta.\)
