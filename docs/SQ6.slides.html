Fact 4.6.4: The Limit Comparison Test

Let \(\displaystyle\sum a_n\) and \(\displaystyle\sum b_n\) be series with positive terms. If

\begin{equation*} \lim_{n \to \infty} \frac{b_n}{a_n} = c \end{equation*}
for some positive (finite) constant \(c\text{,}\) then \(\displaystyle\sum a_n\) and \(\displaystyle\sum b_n\) either both converge or both diverge.

Fact 4.6.10

Supppose we have sequences \(\{a_n\}, \{b_n\}\) so that for some \(k\) we have that \(0\leq b_n\leq a_n\) for each \(k\geq n\) then we have:

  • If \(\displaystyle\sum_{k=n}^\infty a_n\) converges, then so does \(\displaystyle\sum_{k=n}^\infty b_n\text{.}\)
  • If \(\displaystyle\sum_{k=n}^\infty b_n\) diverges, then so does \(\displaystyle\sum_{k=n}^\infty a_n\text{.}\)

Activity 4.6.11

Suppose that you were handed positive sequences \(\{a_n\}, \{b_n\}\text{.}\) For the first few values \(a_n\geq b_n\text{,}\) but after that what happens is unclear until \(n=100\text{.}\) Then for any \(n\geq 100\) we have that \(a_n \leq b_n\text{.}\)

Part 1.

How might we best utilize Fact 4.6.10 to determine the convergence of \(\displaystyle \sum_{n=0}^\infty a_n\) or \(\displaystyle \sum_{n=0}^\infty b_n\text{?}\)

  1. Since \(a_n\) is sometimes greater than, and sometimes less than \(b_n\text{,}\) there is no way to utilize Fact 4.6.10 .
  2. Since initially, we have \(b_n\leq a_n\text{,}\) we can utilize Fact 4.6.10 by assuming \(a_n\geq b_n\text{.}\)
  3. Since we can rewrite \(\displaystyle \sum_{n=0}^\infty a_n=\sum_{n=0}^{99} a_n+\sum_{n=100}^\infty a_n\) and \(\displaystyle \sum_{n=0}^\infty b_n=\sum_{n=0}^{99} b_n+\sum_{n=100}^\infty b_n\) and \(\displaystyle \sum_{n=0}^{99} a_n, \sum_{n=0}^{99} b_n\) are necessarily finite, we can compare \(\displaystyle \sum_{n=100}^\infty a_n, \sum_{n=100}^\infty b_n\) with Fact 4.6.10 .

Fact 4.6.12: The Direct Comparison Test

Let \(\displaystyle\sum a_n\) and \(\displaystyle\sum b_n\) be series with positive terms. If there is a \(k\) such that \(b_n\leq a_n\) for each \(n\geq k\text{,}\) then:

  • If \(\displaystyle \sum a_n\) converges, then so does \(\displaystyle \sum b_n\text{.}\)
  • If \(\displaystyle \sum b_n\) diverges, then so does \(\displaystyle \sum a_n\text{.}\)
