Activity 4.8.3.

Consider the following series:

  1. \(\displaystyle \sum_{n=1}^\infty \frac{n^2(-1)^n}{n^3+1}\text{.}\)

  2. \(\displaystyle \sum_{n=1}^\infty \frac{1}{n^2}\text{.}\)

  3. \(\displaystyle \sum_{n=1}^\infty (-1)^n \left(\frac{2}{3}\right)^n\text{.}\)

  4. \(\displaystyle \sum_{n=1}^\infty -n\text{.}\)

(a)

For each of the series, determine if the series converges using any method.

(b)

For each of the series, compute \(|a_n|\text{.}\)

(c)

For each of the series, determine if the series is absolutely convergent.
