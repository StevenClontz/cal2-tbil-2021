Activity 3.1.10.

Consider the following sequences:

  1. \(\displaystyle\{a_n\}_{n=0}^\infty=\left\{\left(-\frac{2}{3}\right)^n\right\}_{n=0}^\infty\text{.}\)

  2. \(\displaystyle\{b_n\}_{n=0}^\infty=\left\{2\cdot\left(-1\right)^n\right\}_{n=0}^\infty\text{.}\)

  3. \(\displaystyle\{c_n\}_{n=0}^\infty=\left\{-3\cdot \left(1.2\right)^n\right\}_{n=0}^\infty\text{.}\)

(a)

Find the closed form for the \(n\)th partial sum for the following geometric sequences.

  1. \(A_n=\displaystyle\sum_{i=0}^n a_i=\displaystyle\sum_{i=0}^n \left(-\frac{2}{3}\right)^n\text{.}\)

  2. \(B_n=\displaystyle\sum_{i=0}^n b_i=\displaystyle\sum_{i=0}^n 2\cdot\left(-1\right)^n\text{.}\)

  3. \(C_n=\displaystyle\sum_{i=0}^n c_i=\displaystyle\sum_{i=0}^n -3\cdot \left(1.2\right)^n\text{.}\)

(b)

Given the closed forms you found in the previous task, which of the following limits are defined? If defined, what is the limit?

  1. \(\displaystyle\lim_{n\to\infty}A_n\text{.}\)

  2. \(\displaystyle\lim_{n\to\infty}B_n\text{.}\)

  3. \(\displaystyle\lim_{n\to\infty}C_n\text{.}\)
