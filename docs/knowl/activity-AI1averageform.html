  4. \(\displaystyle \int_{\pi}^{4\pi}x\cos(x^2)+x\text{.}\)

  5. \(\displaystyle \frac{1}{3\pi}\int_{\pi}^{4\pi}x\cos(x^2)+x\text{.}\)
