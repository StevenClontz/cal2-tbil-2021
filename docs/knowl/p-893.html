Suppose that \(\displaystyle \sum_{n=0}^\infty a_n=\sum_{n=0}^\infty \frac{1}{n+1}\) which diverges.

  1. Find a sequence \(\{b_n\}\) so that \(0\leq b_n\leq a_n\) for each \(n\) and \(\displaystyle \sum_{n=0}^\infty b_n\) converges.

  2. Find a sequence \(\{b_n\}\) so that \(0\leq b_n\leq a_n\) for each \(n\) and \(\displaystyle \sum_{n=0}^\infty b_n\) diverges.
