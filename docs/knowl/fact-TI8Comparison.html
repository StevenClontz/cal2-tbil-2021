Fact 1.8.28.

Let \(f(x), g(x), h(x)\) be functions such that over a region \(I\text{,}\) \(g(x) < f(x) < h(x)\text{.}\) Then for the integrals \(\displaystyle \int_I f(x) dx, \int_I g(x) dx, \int_I h(x) dx\text{:}\)

  • If \(f(x)\) converges, so does g(x). We need more information to conclude anything about \(h(x)\text{.}\)

  • If \(f(x)\) diverges, so does h(x). We need more information to conclude anything about \(g(x)\text{.}\)
