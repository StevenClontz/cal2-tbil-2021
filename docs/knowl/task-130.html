Task 3.3.15.b.

For which value \(k\) is \(\frac{1}{3n}\leq \frac{1}{2n+3}\) for each \(n\geq k\text{?}\)

  1. \(\frac{1}{3n}\leq \frac{1}{2n+3}\) for each \(n\geq k=0\text{.}\)

  2. \(\frac{1}{3n}\leq \frac{1}{2n+3}\) for each \(n\geq k=1\text{.}\)

  3. \(\frac{1}{3n}\leq \frac{1}{2n+3}\) for each \(n\geq k=2\text{.}\)

  4. \(\frac{1}{3n}\leq \frac{1}{2n+3}\) for each \(n\geq k=3\text{.}\)

  5. There is no \(k\) for which \(\frac{1}{3n}\leq \frac{1}{2n+3}\) for each \(n\geq k\text{.}\)
