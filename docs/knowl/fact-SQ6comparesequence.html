Fact 3.3.11.

Supppose we have sequences \(\{a_n\}, \{b_n\}\) so that for some \(k\) we have that \(0\leq b_n\leq a_n\) for each \(k\geq n\) then we have:

  • If \(\displaystyle_{k=n}^\infty a_n\) converges, then so does \(\displaystyle_{k=n}^\infty b_n\text{.}\)

  • If \(\displaystyle_{k=n}^\infty b_n\) diverges, then so does \(\displaystyle_{k=n}^\infty a_n\text{.}\)
