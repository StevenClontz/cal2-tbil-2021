Activity 1.8.17.

Suppose that \(0< p\) and \(p\neq 1\text{.}\) Consider \(\displaystyle \int \frac{1}{x^p} dx=\frac{1}{(1-p)}x^{1-p}+C\text{.}\)

(a)

If \(p< 1\text{,}\) what is \(\displaystyle \int_0^1 \frac{1}{x^p} dx\text{?}\)

(b)

If \(p> 1\text{,}\) what is \(\displaystyle \int_0^1 \frac{1}{x^p} dx\text{?}\)
