Since \(\displaystyle \sum_{n=0}^\infty \frac{1}{n!}x^n\) converges for each real value \(x\text{,}\) we can define a function \(f(x)=\displaystyle \sum_{n=0}^\infty \frac{1}{n!}x^n\) which assigns to each input \(x\) the limit \(\displaystyle \sum_{n=0}^\infty \frac{1}{n!}x^n\text{.}\)
