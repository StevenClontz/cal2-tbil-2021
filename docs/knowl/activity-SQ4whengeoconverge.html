Activity 4.4.1.

Recall from Section 4.3 that for any real numbers \(a, r\) and \(\displaystyle S_n=\sum_{i=0}^n ar^i\) that:

\begin{align*} S_n=\sum_{i=0}^n ar^i &= a+ar+ar^2+\cdots ar^n\\ (1-r)S_n=(1-r)\sum_{i=0}^n ar^i&= (1-r)(a+ar+ar^2+\cdots ar^n)\\ (1-r)S_n=(1-r)\sum_{i=0}^n ar^i&= a-ar^{n+1}\\ S_n&=a\frac{1-r^{n+1}}{1-r}. \end{align*}

(a)

Using Definition 4.3.11, given each restriction on \(r\text{,}\) determine if \(\displaystyle \sum_{n=0}^\infty ar^n\) converges.

  1. \(r>1\text{.}\)

  2. \(r=1\text{.}\)

  3. \(-1<r<1\text{.}\)

  4. \(r=-1\text{.}\)

  5. \(r<-1\text{.}\)

(b)

For each of the following, if \(\displaystyle \sum_{n=0}^\infty ar^n\) converges, determine what it converges to.

  1. \(r>1\text{.}\)

  2. \(r=1\text{.}\)

  3. \(-1<r<1\text{.}\)

  4. \(r=-1\text{.}\)

  5. \(r<-1\text{.}\)
