Learning Outcomes 2.

How do we use various techniques to integrate less simple functions?

By the end of this chapter, you should be able to...

  1. I can compute the average value of a function on an interval.

  2. I can compute volumes of solids of revolution.

  3. I can compute surface areas of surfaces of revolution.

  4. I can set up integrals to solve problems involving density, mass, and center of mass.

  5. I can set up integrals to solve problems involving work, force, and/or pressure.
