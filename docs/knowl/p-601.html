Fact 2.6.1.

Given a physical object \(m\text{,}\) the work done on that object is

\begin{equation*} W=Fd=mad, \end{equation*}

where \(F\) is the force applied to the object over a distance of \(d\text{.}\) Recall that force \(F=ma\text{,}\) where \(m\) is the mass of the object, and \(a\) is the acceleration applied to it.

Observation 2.6.9. A “how to” for applying integrals to physics.

  1. Estimate the value over a piece of the problem with \(x\) value \(x_i\) over interval of length \(\Delta x\text{.}\)

  2. Find a Riemann sum using (1) which estimates the value in question.

  3. Convert the Riemann sum to an integral and solve.

Fact 2.6.17.

Recall that \(pressure\) is measured as force over area.

\begin{equation*} P=F/A. \end{equation*}

Rewriting this, we have that \(F=PA.\)
