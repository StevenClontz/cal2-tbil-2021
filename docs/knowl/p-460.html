Activity 2.1.4.

Consider a function \(f(x)\) which has average value \(v\) on the interval \([a,b]\text{.}\) Which of the following is most likely always true about \(v\text{?}\)

  1. \(v\) is the average of \(a\) and \(b\text{.}\)

  2. \(v\) is the average of \(f(a)\) and \(f(b)\text{.}\)

  3. \(v\) is the average of \(\min\{f(x)\}\) and \(\max\{f(x)\}\) on the interval \([a,b]\text{.}\)

  4. \(\displaystyle \int_a^b f(x)dx=\int_a^b vdx\text{.}\)

Definition 2.1.5.

Given a function \(f(x)\) defined on \([a,b]\text{,}\) it's average value \(v\) is:

\begin{equation*} v=\frac{1}{b-a}\int_a^b f(x) dx. \end{equation*}
