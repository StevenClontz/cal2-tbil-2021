Activity 3.3.15.

Suppose we wish to determine if \(\displaystyle \sum_{n=1}^\infty \frac{1}{2n+3}\) converged using Fact 3.3.14.

(a)

Does \(\displaystyle \sum_{n=1}^\infty \frac{1}{3n}\) converge or diverge?

(b)

For which value \(k\) is \(\frac{1}{3n}\leq \frac{1}{2n+3}\) for each \(n\geq k\text{?}\)

  1. \(\frac{1}{3n}\leq \frac{1}{2n+3}\) for each \(n\geq k=0\text{.}\)

  2. \(\frac{1}{3n}\leq \frac{1}{2n+3}\) for each \(n\geq k=1\text{.}\)

  3. \(\frac{1}{3n}\leq \frac{1}{2n+3}\) for each \(n\geq k=2\text{.}\)

  4. \(\frac{1}{3n}\leq \frac{1}{2n+3}\) for each \(n\geq k=3\text{.}\)

  5. There is no \(k\) for which \(\frac{1}{3n}\leq \frac{1}{2n+3}\) for each \(n\geq k\text{.}\)

(c)

Use Fact 3.3.14 and compare \(\displaystyle \sum_{n=1}^\infty \frac{1}{2n+3}\) to \(\displaystyle \sum_{n=1}^\infty \frac{1}{3n}\) to determine if \(\displaystyle \sum_{n=1}^\infty \frac{1}{2n+3}\) converges or diverges.
