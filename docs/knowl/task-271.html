Task 4.6.9.e.

Suppose that \(\displaystyle \sum_{n=0}^\infty b_n=\sum_{n=0}^\infty \frac{1}{3^n}\) which converges.

  1. Find a sequence \(\{a_n\}\) so that \(0\leq b_n\leq a_n\) for each \(n\) and \(\displaystyle \sum_{n=0}^\infty a_n\) converges.

  2. Find a sequence \(\{a_n\}\) so that \(0\leq b_n\leq a_n\) for each \(n\) and \(\displaystyle \sum_{n=0}^\infty a_n\) diverges.
