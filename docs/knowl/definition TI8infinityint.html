Definition 1.8.9.

For a function \(f(x)\) and a constant \(c\text{,}\) we let \(\displaystyle \int_c^\infty f(x) dx\) denote

\begin{equation*} \int_c^\infty f(x) dx=\lim_{a\to\infty}\left( \int_c^a f(x) dx\right). \end{equation*}

So just as in limits, this value can be a real number, \(\pm \infty\) or undefined.

Similarly

\begin{equation*} \int_{-\infty}^c f(x) dx=\lim_{a\to-\infty}\left( \int_a^c f(x) dx\right). \end{equation*}
