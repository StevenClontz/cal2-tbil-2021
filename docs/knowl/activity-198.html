Activity 4.1.5.

Consider the sequence \(\displaystyle 1, \frac{1}{3}, \frac{1}{9}, \frac{1}{27}, \ldots\text{.}\) Which of the following choices gives a closed formula for this sequence? Select all that apply.

  1. \(\displaystyle \displaystyle \left\{\left(\frac{1}{3}\right)^{n-1}\right\}_{n=1}^\infty\)

  2. \(\displaystyle \displaystyle \left\{\left(\frac{1}{3}\right)^{n}\right\}_{n=1}^\infty\)

  3. \(\displaystyle \displaystyle \left\{\left(\frac{1}{3}\right)^{n-1}\right\}_{n=2}^\infty\)

  4. \(\displaystyle \displaystyle \left\{\left(\frac{1}{3}\right)^{n+1}\right\}_{n=0}^\infty\)

  5. \(\displaystyle \displaystyle \left\{\left(\frac{1}{3}\right)^{n}\right\}_{n=0}^\infty\)
