(a)

Find \(\int_{\pi/2}^a \cos(x)dx.\)

(b)

Which of the following is true about \(\int_{\pi/2}^\infty \cos(x)dx\text{?}\)

  1. \(\int_{\pi/2}^\infty \cos(x)dx\) is a real number.
  2. \(\int_{\pi/2}^\infty \cos(x)dx\) is positive infinity.
  3. \(\int_{\pi/2}^\infty \cos(x)dx\) is negative infinity.
  4. \(\int_{\pi/2}^\infty \cos(x)dx\) is undefined but neither positive or negative infinity.
