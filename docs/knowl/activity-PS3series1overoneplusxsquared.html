Activity 5.3.4.

Suppose we wished to find the power series for the function \(f(x)=\frac{1}{1+x^2}\text{.}\) Recall that \(g(x)=\frac{1}{1-x}=\displaystyle\sum_{n=0}^\infty x^n.\)

(a)

For what value of \(a\) is \(\frac{1}{1-a}=\frac{1}{1+x^2}\text{?}\)

  1. \(a=x^2\text{.}\)

  2. \(a=-x^2\text{.}\)

  3. \(a=1+x^2\text{.}\)

  4. \(a=1-x^2\text{.}\)

  5. \(a=x^2-1\text{.}\)

(b)

Evaluating \(a\) at the previously found value, which of the following is the power series of \(f(x)=\frac{1}{1+x^2}\text{?}\)

  1. \(f(x)=\displaystyle\sum_{n=0}^\infty (x^2)^{n}=\sum_{n=0}^\infty x^{2n}\text{.}\)

  2. \(f(x)=\displaystyle\sum_{n=0}^\infty (-x^2)^{n}=\sum_{n=0}^\infty (-1)^nx^{2n}\text{.}\)

  3. \(f(x)=\displaystyle\sum_{n=0}^\infty (1+x^2)^n\text{.}\)

  4. \(f(x)=\displaystyle\sum_{n=0}^\infty (1-x^2)^n\text{.}\)

  5. \(f(x)=\displaystyle\sum_{n=0}^\infty (x^2-1)^n\text{.}\)

(c)

Given that the interval of convergence for \(a\) is \(-1 < a < 1\text{,}\) what is the interval of convergence for \(x\text{?}\)

  1. \(-1 < x < 1\text{.}\)

  2. \(-1 < x < 0\text{.}\)

  3. \(0 < x < 1\text{.}\)

  4. \(0 < x < 2\text{.}\)

  5. \(-2 < x < 0\text{.}\)

  6. \(-2 < x < 2\text{.}\)
