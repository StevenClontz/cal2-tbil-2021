Activity 5.3.5.

What function \(f(x)\) has power series \(f(x)=\displaystyle \sum_{n=0}^\infty \frac{(-1)^nx^n}{n!}=1-x+\frac{x^2}{2}-\frac{x^3}{6}+\cdots\text{?}\)

  1. \(f(x)=(-1)^ne^x\text{.}\)

  2. \(f(x)=e^{-1}\text{.}\)

  3. \(f(x)=-e^x\text{.}\)

  4. \(f(x)=e^{-x}\text{.}\)

  5. \(f(x)=-e^{-x}\text{.}\)
