Part 5.

Note that

\begin{align*} \sqrt{(\Delta x)^2+(f(x_i+\Delta x)-f(x_i))^2} & = \sqrt{(\Delta x)^2\left(1+\left(\frac{f(x_i+\Delta x)-f(x_i)}{\Delta x} \right)^2\right)}\\ &=\sqrt{1+\left(\frac{f(x_i+\Delta x)-f(x_i)}{\Delta x} \right)^2}\Delta x \end{align*}

Which of the following best describes \(\displaystyle\lim_{\Delta x\to 0} \frac{f(x_i+\Delta x)-f(x_i)}{\Delta x}\text{?}\)

  1. \(0\text{.}\)
  2. \(1\text{.}\)
  3. \(f'(x_i)\text{.}\)
  4. This limit is undefined.

Fact 2.2.2

Given a differentiable function \(f(x)\text{,}\) the arclength of \(y=f(x)\) defined on \([a,b]\) is computed by the integral:

\begin{align*} \lim_{n\to \infty}\sum \sqrt{(\Delta x)^2+(f(x_i+\Delta)-f(x_i))^2} & =\lim_{n\to \infty}\sum \sqrt{1+\left(\frac{f(x_i+\Delta x)-f(x_i)}{\Delta x} \right)^2}\Delta x\\ & = \int_a^b \sqrt{1+(f'(x))^2}dx. \end{align*}
