Section 4.2 Sequence Properties and Limits (SQ2)

Learning Outcomes

  • Determine if a sequence is convergent, divergent, monotonic, or bounded, and compute limits of convergent sequences.

Activity 4.2.1.

We will consider the fnuciton \(\displaystyle \frac{4x+8}{x}\text{.}\)

(a)

Compute the limit \(\displaystyle \lim_{x\to \infty} \frac{4x+8}{x}\text{.}\)

  1. \(\displaystyle 0\)

  2. \(\displaystyle 8\)

  3. \(\displaystyle 1\)

  4. \(\displaystyle 4\)

(b)

What value does \(f(x)\) converge to as \(x\to\infty\text{?}\)

  1. \(\displaystyle 0\)

  2. \(\displaystyle 8\)

  3. \(\displaystyle 1\)

  4. \(\displaystyle 4\)

(c)

Determine on which intervals \(f(x)\) is increasing and/or decreasting.

(d)

Which statement best describes \(f(x)\) for \(x>0\text{?}\)

  1. \(f(x)\) is bounded above by 4

  2. \(f(x)\) is bounded below by 4

  3. \(f(x)\) is bounded above and below by 4

  4. \(f(x)\) is not bounded above

  5. \(f(x)\) is not bounded below

(e)

Determine on which intervals \(f(x)\) is concave up and/or concave down.

(f)

Sketch a graph of the function \(f(x)\text{.}\)
